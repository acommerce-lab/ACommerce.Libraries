# بَذر مَدارِس "عشير القديم"

مَشروع console مُنفَصِل يَعمَل مُباشَرَةً عَلى قاعِدَة بَيانات **عشير القديم في
الإنتاج** (مُستودَع `acommerce-lab/ACommerce.Libraries`). يَقوم بِـ:

1. **اشتِقاق المالِك**: يَختار تِلقائيّاً المُستخدِم الأَكثَر نَشراً لِلعُروض
   (أَكثَر `VendorId` تَكراراً في `ProductListing` غير المَحذوف) — أو يُمكِن
   تَحديده يَدويّاً عَبر `Seeder:OwnerProfileId`.
2. **تَعطيل القَديم (حَذف ناعِم)**: يَضبِط `IsDeleted=true` (+`IsActive=false`)
   عَلى كُلّ الفئات القَديمَة + رَوابِطها + الخصائص القَديمَة الخالِصَة. يُبقي
   الخصائص الأَساسيّة المُشتَرَكَة (`title/description/price/city/location/
   images/contact`) فَعّالَة لِأَنّ المَدارِس تُعيد استِخدامها. **لا حَذف
   نِهائيّ.**
3. **بَذر تَصنيف المَدارِس الاحترافيّ**: فئات (مدارس للاستثمار / للمشاركة /
   للإيجار) + ١٩ خاصّيّة احترافيّة (نوع المدرسة، المنهج، السعة، الترخيص،
   المرافق، نوع الفرصة الاستثماريّة، الحصّة، الإيراد…) + رَبطها بِالفئات.
4. **بَذر عُروض مَدارِس عَيِّنَة**: عُروض واقِعيّة بِأَسعار استثمار + مَرافِق +
   ملاعب + صُوَر تُنَزَّل مِن الويب → تُرفَع عَبر `IStorageProvider` → تُخزَّن
   روابِطها مَع العَرض. مَملوكَة لِلمالِك المُشتَقّ.

## الأمان

- **DRY-RUN افتراضيّ**: التَّشغيل بِلا مُتغَيِّر بيئة يَطبَع كُلّ ما سَيَفعَل
  دون كِتابَة أَيّ شَيء.
- **التَّطبيق الفِعليّ**: يَتَطَلَّب `SEED_APPLY=true` صَريحاً.
- كُلّ الكِتابَة داخِل **transaction واحِدَة** (عَبر execution strategy لِأَنّ
  DbContext الإنتاج يُفَعِّل `EnableRetryOnFailure`) — إمّا تَنجَح كُلّها أو
  تَتَراجَع.
- **idempotent**: يَفحَص الوُجود بِالـ Id/Code قَبل أَيّ إدراج.
- **نَسخَة احتياطيّة**: يُصَدِّر الفئات/الخصائص/الرَّوابِط إلى
  `backup-taxonomy-*.json` قَبل التَّعطيل.

## المَكان والتَّشغيل

> صَلاحيّات الوَكيل في GitHub مَحصورَة بِـ `acommerce-platform`، لِذا المَصدَر
> مَحفوظ هُنا. لكِنّ المَشروع يُشير إلى مَشاريع **عشير القديم**
> (`ACommerce.Libraries`) الَّتي ليست في هذا المُستودَع — فَيَجِب أَن تَملُك
> نُسخَة مُستَنسَخَة مِنها.

يَحُلّ المَشروع جَذر عشير القديم تِلقائيّاً عَبر الخاصّيّة `AshareLegacyRoot`،
وَالافتراضيّ يَفتَرِض **نُسخَة جَنباً إلى جَنب** (وهو الأكثر شُيوعاً):

```
…\source\repos\acommerce-platform\Tools\AshareLegacySchoolSeeder   ← هذا المَشروع
…\source\repos\ACommerce.Libraries                                 ← عشير القديم
```

يَعمَل الافتراضيّ سَواء بَقِيَ المَشروع في `acommerce-platform\Tools\` (مَع
`ACommerce.Libraries` شَقيقَة) **أو** نُقِلَ إلى `ACommerce.Libraries\Tools\`.

```bash
# 1) استَنسِخ عشير القديم (لو لَم يَكُن مَوجوداً)
git clone https://github.com/acommerce-lab/ACommerce.Libraries

# 2) جَهِّز الإعدادات (لا تَرفَعها — appsettings.Local.json مُتجاهَل في git)
cp appsettings.Local.example.json appsettings.Local.json
#   املأ: ConnectionStrings:DefaultConnection (الإنتاج)،
#         HostSettings:BaseUrl، وقِسم Files:Storage:* لِلتَّخزين.

# 3) مُعايَنَة أوّلاً (لا كِتابَة)
dotnet run

# 4) التَّطبيق الفِعليّ بَعد مُراجَعَة المُعايَنَة
SEED_APPLY=true dotnet run
```

لَو كانَ مُستودَع عشير القديم في مَسار مُختَلِف، مَرِّره صَراحَةً:

```bash
dotnet run -p:AshareLegacyRoot=D:\path\to\ACommerce.Libraries
# (أو اضبط مُتغَيِّر بيئة AshareLegacyRoot)
```

لَو لَم يُعثَر عَلى المَشاريع، يُعطي البِناء رِسالَة واضِحَة تُرشِدك لِضَبط
`AshareLegacyRoot`.

## مُلاحظات

- **الصُّوَر**: روابِط Unsplash عامّة مَوضوعَة في `SchoolSeedData.Samples()`.
  راجِعها أو استَبدِلها قَبل التَّشغيل الإنتاجيّ. لَو فَشَل تَنزيل رابِط، يُتَخَطّى
  بِأَمان.
- **التَّخزين**: يَجِب أَن يُطابِق مُزَوِّد الإنتاج (GCS أو Aliyun) لِتَكون
  روابِط الصُّوَر مَقروءَة في التَّطبيق. بِلا إعداد تَخزين يَستَخدِم تَخزيناً
  محليّاً (لِلتَّجرِبَة فَقَط).
- **GUIDs**: تَصنيف المَدارِس يَستَخدِم نَطاق `aa*/bb*/cc*` المَعزول عَن نَطاق
  البَذر القَديم (`1*/2*/3*/4*`) فَلا تَصادُم.
