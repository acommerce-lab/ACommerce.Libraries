# Rukkab Rider — Blazor PWA (thin shell)

This project is a minimal Blazor WebAssembly PWA that references the shared UI library `libs/frontend/Rukkab.UI`.

How to run (development):

```bash
cd Apps/Rukkab/Rider/Rider.Web.Blazor
dotnet run
```

The app expects the backend Rider API available at `http://127.0.0.1:5001`.
